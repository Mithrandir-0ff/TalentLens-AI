# TalentLens-AI
TalentLens-AI is AI-powered football scouting platform for identifying talent through semantic analysis of player statistics. Built with Python, FastAPI, and LLM structured output.

# Отчет по Заданию №1: Прототип API TalentLens-AI

1. Реализован функционал(код)
FastAPI Сервер: Развернут базовый эндпоинт для приема запросов на анализ игроков.
Инструменты агенты(Tools): Реализовано несколько нужных инструментов для поиска и анализа игроков для агента.(поиск по id, подробная статистика и тд)
Интеграция LLM: Создан LangChain-агент на базе модели "gpt-oss-120b" через LiteLLM с температурой 0 для стабильности.
Структурированный вывод: Настроен структурированный вывод JSON.

Дополнения:
Используется RapidAPI, а конкретно SofaScore для работы с футбольной статистикой
Реализован прототип работы агента, способный по запросу на игрока выдать его статистику и рекомендации
Начата реализация дополнительных и более сложных функций
