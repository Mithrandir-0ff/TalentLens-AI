# TalentLens-AI
TalentLens-AI is AI-powered football scouting platform for identifying talent through semantic analysis of player statistics. Built with Python, FastAPI, and LLM structured output.
TalentLens-AI is a platform for football scouting and player identification based on the semantic analysis of various statistics. It is built using Python and FastAPI, utilizing an LLM based on the gpt-oss-120b model, which uses the SportAPI as its statistical database.

Users can make requests through prompts based on their specific required criteria. The model then searches for matching players by ID in the SportAPI, provides the relevant player(s) with all necessary statistical data, and generates a text response describing the reason for selecting a particular player, along with a brief analysis of their statistical data. The analysis can be either concise or detailed, depending on the user's request—by default, it provides a brief statistical analysis and a conclusion on how well the player fits the user's needs and their effectiveness in football terms.

TalentLens-AI can function as a chat with persistent memory in cases of, for example, imprecise queries. The model can ask clarifying questions to provide a more accurate final answer. For overly broad questions, such as "I need a male player," the model will clarify the details of the request to achieve a precise result, as providing 20,000 players at once would be extremely problematic. Additionally, for irrelevant questions, commands, or nonsense from the user, the model will determine that the query is outside its scope of work and provide standard responses like: "The question is not within my specialty; please clarify what you would like to find in the world of talents!"

The main goal of the project is to simplify the player search process, not only for professional football scouts but also, for example, for Football Manager or EA FC players looking for recruits for their virtual teams. Thanks to TalentLens-AI, users no longer need to scour statistical websites or search for articles and player descriptions themselves—TalentLens-AI can do it all in just a few prompts!
